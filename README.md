# DevFusion Website

Static multi-page website for Team DevFusion.

## Pages

- Home: `index.html`
- About: `about.html`
- Projects: `projects.html`
- Contact: `contact.html`

## Notes

- Team data, project data, and contact links live in `site-data.js`.
- The contact form opens a mail draft to `devfusionorg@gmail.com`.
- Project cards support live links and APK links when you add them later.
